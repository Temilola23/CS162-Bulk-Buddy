# Bulk Buddy -- Frontend

The frontend for Bulk Buddy, built with React.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Structure

```
frontend/
├── src/
│   ├── components/        # Shared React UI pieces
│   ├── data/              # Mock data for MVP screens
│   └── pages/             # Route-level screens
├── index.html
├── package-lock.json
├── package.json
└── vite.config.js
```

Design assets and Figma exports are documented under `docs/design/`, not `frontend/public/`.

## Running Locally

```bash
cd frontend
npm run dev
```

## Testing

Build the frontend:

```bash
cd frontend
npm run build
```
